# 🎓 College Admission Assistant (RAG Chatbot)

This is a fully functional, memory-aware conversational AI assistant designed to handle college admission and campus-related queries. Built using a **Retrieval-Augmented Generation (RAG)** architecture, it eliminates AI hallucinations by pulling answers strictly from official university PDFs and live website URLs.

## 🚀 Features
* **Zero Hallucinations:** Answers strictly from official context.
* **Conversational Memory:** Remembers previous chat history.
* **Local Embeddings:** Bypasses API rate limits using HuggingFace `all-MiniLM-L6-v2`.
* **Multi-Modal:** Reads both PDFs and live web links.
